ServiceTop
Интерактивный TUI-менеджер сервисов OpenRC для Gentoo Linux.

ServiceTop — это удобная консольная утилита, которая помогает управлять системными службами OpenRC, не переключаясь постоянно между терминалом и man-страницами. Всё под рукой: статус, автозапуск, мониторинг системы и логи в одном окне.

Основные возможности
Управление сервисами: Запуск, остановка, перезапуск и zap одной кнопкой.

Управление автозапуском: Быстрая настройка rc-update через удобное меню.
Мониторинг: Встроенный мониторинг CPU, RAM и Load Average.
TOP процессов: Список самых ресурсоемких процессов прямо в интерфейсе.
Умный поиск: Быстрая фильтрация списка служб.
Гибкая настройка: Поддержка Избранного (Favorites), Черного списка (Blacklist) и приоритетов отображения.
Безопасность: Защита критически важных сервисов (например, sshd, net.*) от случайного выключения.
Локализация: Поддержка RU/EN с переключением на лету.

Установка
Для работы скрипта требуются права root. Рекомендуется использовать Python 3.9+.

Bash
# Создание директории
sudo mkdir -p /opt/servicetop/

# Скачивание скрипта
sudo curl -o /opt/servicetop/servicetop.py https://raw.githubusercontent.com/PEAKT0P/servicetop.py/main/servicetop.py

# Установка прав и создание симлинка
sudo chmod +x /opt/servicetop/servicetop.py
sudo ln -sf /opt/servicetop/servicetop.py /usr/local/bin/servicetop
Использование
Просто запустите команду от суперпользователя:

Bash
sudo servicetop
Горячие клавиши
/ — Поиск сервисов
G — Переключение языка (RU/EN)
I — Информационная панель о выбранном сервисе
F — Добавить/удалить из избранного
B — Добавить/удалить из черного списка
P — Настройка приоритета

F10 / Q — Выход

Требования
Система с OpenRC.

psutil (опционально, для более точного мониторинга ресурсов; если отсутствует, используется стандартный fallback через /proc/).
